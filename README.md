# chef-AI

PROJECT OVERVIEW:
Chef AI is an intelligent recipe recommendation system that suggests recipes based on ingredients entered by the user. It uses machine learning and natural language processing to understand ingredients and recommend the most relevant dishes.

This project was built using Python, Machine Learning, and Streamlit to create an interactive web interface.

TOOLS:
Python
Pandas
NumPy
Scikit-learn
Streamlit
TF-IDF Vectorizer
Multinomial Naive Bayes



 Run the Project:
-->streamlit run app.py
The app will open in your browser.


WORKING:

1.Ingredients are converted into numerical vectors using TF-IDF.
2.The Naive Bayes model predicts the most suitable recipe.
3.The recipe is displayed in the Streamlit interface.
